# Recruiting Pipeline — Aleut Federal

> Aleut Federal company context lives in `ALEUT-FEDERAL-CONTEXT.md` (uploaded as `00-ALEUT-FEDERAL-CONTEXT.md`).

Help manage the recruiting pipeline from sourcing through offer acceptance for federal-contractor roles.

## Aleut Federal Context — Federal-Contractor Recruiting

- **OFCCP / EO 11246 / VEVRAA / Section 503** — AAP recruitment, posting, applicant-tracking, and recordkeeping obligations where applicable (contract ≥ $50K, headcount ≥ 50).
- **Pay transparency** — comply with state laws (CA, CO, IL, MD, NY, WA) for posting locality.
- **Clearance pipeline** — track cleared-candidate channels; long-lead investigations start at offer, not start date.
- **Citizenship** — when required, posting language must comply with IRCA non-discrimination.
- **Federal-program experience** — capture as a structured field for past-performance matching to opportunities.
- **PIA / source-selection screen** for candidates from competitors or recent federal employees.
- **VETS-4212 / EEO-1** annual reporting — track recruitment outcomes by veteran / disability / EEO categories.
- **Mentor-Protégé** — protégé personnel into JV roles must comply with the SBA-approved MPA.

Stages: Sourcing → Application → Screening (compliance + competency) → Interview → Offer (contingent) → Onboarding (E-Verify, drug screen, background, clearance passback).

## Pipeline Stages

| Stage | Description | Key Actions |
|-------|-------------|-------------|
| Sourced | Identified and reached out | Personalized outreach |
| Screen | Phone/video screen | Evaluate basic fit |
| Interview | On-site or panel interviews | Structured evaluation |
| Debrief | Team decision | Calibrate feedback |
| Offer | Extending offer | Comp package, negotiation |
| Accepted | Offer accepted | Transition to onboarding |

## Metrics to Track

- **Pipeline velocity**: Days per stage
- **Conversion rates**: Stage-to-stage drop-off
- **Source effectiveness**: Which channels produce hires
- **Offer acceptance rate**: Offers extended vs. accepted
- **Time to fill**: Days from req open to offer accepted
